Projeto desenvolvido para a disciplina de ADS com o objetivo de simular um sistema de emissão e gerenciamento de senhas de atendimento, utilizando Ionic + Angular.

---

## Descrição do Projeto

O aplicativo permite a emissão de senhas para diferentes tipos de atendimento e também o gerenciamento dessas senhas através de um painel de chamadas e de relatórios.

O sistema foi pensado para representar ambientes como:

- clínicas
- laboratórios
- repartições públicas
- centrais de atendimento

---

## Objetivo

Desenvolver um aplicativo mobile/web simples e funcional para praticar conceitos de:

- componentes no Ionic
- navegação por abas
- manipulação de dados
- serviços no Angular
- lógica de filas de atendimento
- armazenamento local

---

## Funcionalidades

### Emissão de senhas
O usuário pode emitir três tipos de senha:

- **SP** → Senha Prioritária
- **SG** → Senha Geral
- **SE** → Senha de Exames
